### [Cyber Tales AI](https://github.com/codewithishita/cyber-tales-ai) — AI-Powered Cybersecurity Learning Platform
A full-stack, production-grade web app that teaches children cybersecurity through AI-generated interactive stories and challenges.

- GPT-4o generates unique branching narratives — every playthrough is different
- Streaming AI mentor (SSE) with full user context and conversation memory
- Interactive challenge simulator: phishing detection, URL threat analysis, password strength evaluation
- XP system, 6 rank tiers, achievement badges, and a live Recharts dashboard
- PostgreSQL schema with append-only XP event log for time-series analytics

`Next.js 15` `TypeScript` `React 18` `OpenAI GPT-4o` `PostgreSQL` `Prisma` `Supabase` `Clerk` `Tailwind CSS` `Vercel`
